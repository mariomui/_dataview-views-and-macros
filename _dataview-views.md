## About

This [[folder-page,vis-Noteshippo,]] describes the files that should be housed inside this folder. It describes the types of files that should be classified as a [[Partial-dataview,vis-Noteshippo,]] , or a [[interim--macro-inserting-private-lc-citum]]. Mostly, the files are a sandbox for pages with views powered by [[Dataview-plugin,bt.-Obsidianmd-app,]].

Any subfolders will also be described here.

Static pages are post processed and will be shared.

The code stays off the primary note but the transcluded preview is available. This works well for dynamically generate hotkey pages.
